# 📋 Task Planner Pro

A modern, responsive Task Planner web application built using **HTML**, **CSS**, and **JavaScript**. It helps users efficiently organize their daily tasks with features like priority management, due dates, progress tracking, dark mode, and persistent local storage.

## ✨ Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑 Delete tasks
- ✔ Mark tasks as completed
- 🔍 Search tasks instantly
- 🚩 Priority levels (High, Medium, Low)
- 📂 Task Categories (Study, Work, Personal, Shopping, Fitness)
- 📅 Due date selection
- 📊 Progress bar
- 📈 Task statistics
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
- 🌙 Dark Mode
- 💾 Local Storage (tasks remain after page refresh)
- ⌨ Press **Enter** to quickly add tasks
- 📱 Responsive design for desktop and mobile devices

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/pragatibhandare05-ux/Task-Planner.git
```

### 2. Open the Project

Open the project folder in **Visual Studio Code** or any code editor.

### 3. Run the Application

Open the `index.html` file in your preferred web browser.

No additional installation or dependencies are required.

## 💻 How to Use

1. Enter your task.
2. Select the priority level.
3. Choose a due date.
4. Click **Add Task**.
5. Mark tasks as complete using the checkbox.
6. Edit or delete tasks when needed.
7. Use the search bar to quickly find tasks.
8. Toggle Dark Mode for a different appearance.
